# Project: Awan Whiteboard Plugin

## Overview

Build a production-ready plugin called **Awan Whiteboard** for the AwanTools platform.

This is NOT a standalone application.

It must be built as a modular plugin that integrates into an existing PHP-based platform called **AwanTools**.

The plugin should combine the best features of:

* Microsoft Whiteboard
* Nebo
* Excalidraw
* Jamboard
* Obsidian Canvas

while remaining lightweight and deployable on shared hosting or low-cost VPS infrastructure.

The plugin must be designed for:

* Students
* Teachers
* Tutors
* Study groups
* Educational institutions
* Team collaboration

The system must prioritize free and open-source technologies.

No paid APIs should be required.

# Core Feature 5: Handwriting Recognition

This is the signature feature.

Users write using:

* Mouse
* Stylus
* Touch

System captures stroke vectors.

Recognition Pipeline:

Step 1:
Store stroke data.

Step 2:
Render selection to temporary image.

Step 3:
Run OCR.

Preferred OCR options:

1. Tesseract.js
2. Tesseract OCR
3. PaddleOCR

Must remain free.

No paid APIs.
